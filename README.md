🎵 1. Project Title / Headline
🎧 Spotify Analytics: Global Top 50 Music Dashboard

An interactive Power BI dashboard built to analyze Spotify’s Top 50 chart data—focusing on song rankings, artist dominance, popularity trends, album characteristics, and listener preferences.

📌 2. Short Description / Purpose

The Spotify Analytics Dashboard is a dynamic and visually engaging Power BI report designed to explore and analyze Spotify Top 50 music data.

It enables users to track chart performance, evaluate artist popularity, analyze explicit content trends, and compare album vs single performance—helping uncover insights into global streaming behavior and music consumption patterns.

This tool is intended for music analysts, streaming platforms, entertainment businesses, and data-driven decision-makers.

🛠 3. Tech Stack

The dashboard was built using the following tools and technologies:

• 📊 Power BI Desktop – Primary data visualization platform
• 📂 Power Query – Data cleaning, transformation, and preprocessing
• 🧠 DAX (Data Analysis Expressions) – KPIs, ranking measures, time intelligence, and dynamic calculations
• 🧩 Data Modeling – Structured relationships and optimized schema for efficient cross-filtering
• 📁 File Formats – .pbix for development and .png for dashboard previews

📂 4. Data Source

Source: Spotify Top 50 World Dataset

The dataset contains information about globally trending songs, including:

Chart Date

Song Name

Artist

Position (Rank)

Popularity Score

Duration (ms)

Album Type (Single / Album)

Total Tracks

Release Date

Explicit Content Indicator

The dataset enables time-based, performance-based, and content-based analysis of streaming trends.

🚀 5. Features / Highlights
• Business Problem

The music streaming industry generates massive volumes of data daily. However, understanding:

Which artists dominate global charts?

Do explicit songs perform better?

Are singles outperforming albums?

How long do songs stay on charts?

What duration range performs best?

… is difficult without structured analytics.

• Goal of the Dashboard

To deliver an interactive analytical tool that:

Tracks song ranking trends over time

Identifies top-performing artists

Analyzes popularity distribution

Evaluates explicit content impact

Compares album vs single performance

Provides insights into music consumption behavior

• Walkthrough of Key Visuals
🔑 Key KPIs (Top Section)

Total Songs

Distinct Artists

Average Popularity

Best Chart Position

% Explicit Songs

Average Song Duration

📈 Ranking Trend (Line Chart)

Displays movement of song positions over time to identify rising hits and declining tracks.

🎤 Top Artists (Bar Chart)

Number of chart appearances

Average popularity score

🔞 Explicit vs Non-Explicit Analysis (Column Chart)

Compares:

Count of explicit vs clean songs

Average popularity by content type

💿 Album Type Distribution (Donut / Bar Chart)

Analyzes:

Singles vs Albums performance

Average tracks per album

Popularity comparison

• Business Impact & Insights

📊 Trend Analysis: Identify emerging music trends and viral patterns.

🎯 Content Strategy: Understand impact of explicit content on popularity.

🎵 Artist Performance Tracking: Evaluate consistency and dominance of artists.

💼 Industry Application: Useful for music labels, streaming platforms, marketing teams, and entertainment analysts to make data-driven decisions.

📈 Strategic Insights: Detect optimal song duration, album strategies, and listener behavior patterns.
